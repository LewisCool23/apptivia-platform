# Apptivia Backend

A Node.js backend for Apptivia with Express, Socket.io, Supabase, authentication, and REST API.

## Setup

1. Copy `.env.example` to `.env` and fill in your Supabase and JWT credentials.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start` or `npm run dev` (for auto-reload).

## Features
- REST API (user registration/login)
- Socket.io real-time events
- Supabase integration
- JWT authentication
